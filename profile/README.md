# Sookmyung APPS

숙명여자대학교 소프트웨어학부 모바일 앱 프로그래밍 동아리

</div>

## WHAT IS?

APPS는 ‘모바일 앱 개발’이라는 공통분모를 가진 학생들이 모여 스터디, 프로젝트, 공모전, 전시회 등 다양한 활동을 통해 자신을 성장시키고,
재밌고 유의미한 앱 및 웹 서비스를 개발하는 소프트웨어학부 소속 학회입니다.
